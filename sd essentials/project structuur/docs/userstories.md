Als student wil ik het PVE (Programma van Eisen) doornemen zodat ik weet wat de opdracht vereist.
Als student wil ik verschillende design ideeën schetsen zodat mijn portfolio uniek en persoonlijk wordt.
Als student wil ik feedback krijgen op mijn ontwerp zodat ik weet of ik op de goede weg ben.
Als student wil ik diverse media verzamelen zodat het project er verzorgd en professioneel uitziet.
Als student wil ik de HTML en CSS opdrachten en lessen gevolgd en  gemaakt hebben zodat ik meer weet over HTML en CSS.
Als student wil ik de HTML Grid en Flexbox opdrachten maken zodat ik een nette layout kan maken in HTML en CSS.
Als student wil ik een projectstructuur in elkaar zetten zodat ik overzicht heb en alles kan terugvinden.
Als student wil ik een projectenpagina maken zodat bezoekers kunnen zien wat ik gemaakt heb en waar ik goed in ben.
Als student wil ik een homepage voor mijn portfolio bouwen zodat mijn bezoekers meteen een goede indruk krijgen.
Als student zorg ik dat het porfolio online staat en benaderbaar is via de school VPS
